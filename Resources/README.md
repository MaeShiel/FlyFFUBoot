# Resources Folder

## App Icon
Place your application icon file here as `app.ico`

### Icon Requirements:
- Format: .ico file (not PNG or JPG)
- Recommended sizes in .ico: 16x16, 32x32, 48x48, 256x256
- If you have a PNG, convert it to .ico using online tools like:
  - https://convertio.co/png-ico/
  - https://www.icoconverter.com/

### File Name:
- `app.ico` - This is the icon that will be used for your application

Once you place the icon file here, rebuild the project to see it applied.
